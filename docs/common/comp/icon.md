# 字体图标 

> Symbol 采用SVG方式渲染

示例：
```
<mIcon iconClass="iconName" className="md-icon" />
```


图标查询：
```
src\assets\font\demo_index.html
```

> props Attributes

|参数|说明|类型|可选值|默认|
| ------ | ------ | ------ |------ |------ |
| iconClass | 图标名称 | String | 必填 | 未命名 |
| className | 样式名称 | String | — | 无 |
